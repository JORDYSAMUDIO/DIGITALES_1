# CALCULADORA Y PROYECTO FINAL
En este repositorio se encuentra la documentación de el proyecto final y el de la calculadora de numeros binarios. Cada carpeta contiene los respectivos diagramas de flujo , de caminos de datos y de estados para cada uno de las operaciones de la calculadora (multiplicación, división, raiz, binario a BCD), y del proyecto final qu es la implementacion de modulos de control a la Colorlight para mostrar en pantalla un grupo de imagenes contenidad en memoria de la misma pantalla.
